# 🌐 Portafolio Profesional | Stefani Sánchez

Bienvenido/a a mi portafolio web ✨  
Este proyecto representa mi identidad como desarrolladora y reúne mis habilidades, proyectos y experiencia en desarrollo web.

## 📌 Descripción

Este portafolio fue diseñado para mostrar de forma clara, moderna y atractiva mi perfil profesional. Incluye diferentes secciones donde presento información relevante sobre mí, mis conocimientos y los proyectos que he desarrollado.

## 🧩 Secciones del portafolio

- 🏠 Inicio  
- 👩‍💻 Sobre mí  
- 🛠️ Habilidades  
- 📁 Proyectos  
- 📞 Contacto  

## 🚀 Tecnologías utilizadas

- HTML5  
- CSS3  
- Animaciones con CSS  

## 🎨 Características

- Diseño visualmente atractivo  
- Interfaz limpia y organizada  
- Navegación intuitiva  
- Animaciones suaves  
- Adaptable a diferentes dispositivos  
- Código fácil de entender

## 🔧 Instalación y uso

1. Clona este repositorio:  
git clone https://github.com/tu-usuario/tu-repositorio.git  

2. Ingresa a la carpeta del proyecto:  
cd tu-repositorio  

3. Abre el archivo `index.html` en tu navegador.
